# Project Title

Argent Bank App.

## Requirements

- Node.js version 16.18.1
- npm version 8.19.2

## Installation

To install the project dependencies, run the following command:

```bash
npm i
```

## Usage

To launch the app in development mode, run the following command:

```bash
npm run dev
```

## Environment

This project uses Vite for the environment.
